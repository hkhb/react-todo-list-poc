# ReactTodoList

DEMOのリンクhttps://hkhb.github.io/react-todo-list-poc/

## 概要
ReactとTypeScriptで実装したシンプルなTodoアプリです。  
タスクの追加・編集・削除ができ、モーダルによるUI操作を体験できます。

## 機能一覧
- タスク（リスト）の追加
- タスクの編集（モーダルUI）
- タスクの削除
- 作成・編集時間の表示（date.fns使用）
- ローカルストレージに保存できるようにする。

## 使用技術
### フロントエンド
- HTML/CSS
- TypeScript (ES6)
- React

### その他
- GitHub Actions
- date.fns（日付処理ライブラリ）
